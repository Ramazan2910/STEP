
Задание 1: Подсчет гласных в строке
Напишите программу, которая запрашивает у пользователя строку и подсчитывает количество гласных букв в этой строке.

Задание 2: Обратный порядок слов
Напишите программу, которая запрашивает у пользователя строку и выводит слова в обратном порядке.


Задание 3: Факториал числа
Напишите программу, которая запрашивает у пользователя целое число и вычисляет его факториал с использованием цикла.

Задание 4: Подсчет слов в строке
Напишите программу, которая запрашивает у пользователя строку и подсчитывает количество слов в этой строке.


Задание 5: Умножение всех элементов списка на заданное число
Напишите программу, которая запрашивает у пользователя числа через пробел, сохраняет их в список и умножает каждый элемент списка на заданное число, введенное пользователем.

Задание 6: Сумма элементов списка
Напишите программу, которая запрашивает у пользователя числа через пробел, сохраняет их в список и выводит сумму всех элементов списка.

Задание 7: Удаление элемента из списка
Напишите программу, которая запрашивает у пользователя числа через пробел, сохраняет их в список, а затем запрашивает число для удаления и удаляет его из списка.

Задание 8: Количество элементов в списке
Напишите программу, которая запрашивает у пользователя числа через пробел, сохраняет их в список и выводит количество элементов в списке.

Задание 9: Среднее значение элементов списка
Напишите программу, которая запрашивает у пользователя числа через пробел, сохраняет их в список и выводит среднее значение этих чисел.


Задание 10: Есть некоторый текст. Посчитайте в этом тексте количество предложений и выведите на экран полученный
результат.


Задание 11
Два списка целых заполняются случайными числами.
Необходимо:
■ Сформировать третий список, содержащий элементы
обоих списков;
■ Сформировать третий список, содержащий элементы
обоих списков без повторений;
■ Сформировать третий список, содержащий элементы
общие для двух списков;
■ Сформировать третий список, содержащий только
уникальные элементы каждого из списков;
■ Сформировать третий список, содержащий только
минимальное и максимальное значение каждого из
списков.


Task 12: Проверка принадлежности к диапазону
Напишите программу, которая запрашивает у пользователя целое число и проверяет, находится ли оно в диапазоне от 10 до 20 включительно.


Task 13: Проверка делимости
Напишите программу, которая запрашивает у пользователя целое число и проверяет, делится ли оно на 3 и на 5 одновременно.


Task 14: Определение времени года
Напишите программу, которая запрашивает у пользователя номер месяца (1-12) и выводит название соответствующего времени года.


Task 15: Проверка типа треугольника
Напишите программу, которая запрашивает у пользователя длины трех сторон треугольника и определяет, является ли треугольник равносторонним, равнобедренным или разносторонним.


Task 16: Определение времени суток
Напишите программу, которая запрашивает у пользователя время в формате 24-часов (целое число от 0 до 23) и выводит, является ли это утро, день, вечер или ночь.

Task 17: Проверка квадрата
Напишите программу, которая запрашивает у пользователя длины двух сторон прямоугольника и определяет, является ли этот прямоугольник квадратом.
