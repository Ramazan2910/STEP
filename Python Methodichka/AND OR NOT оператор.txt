############### AND OR NOT операторы

Эти операторы возвращают значение bool True/False.
AND и OR операторы используются между двумя операндами или
двумя выражениями и могут использоваться в неограниченном количестве.
#--------------AND------------

AND оператор возвращает True только если оба операнда возвращают True.
В противном случае возвращает False.
#--------------OR------------

OR оператор возвращает True, если хотя бы один из операндов возвращает True.
Если оба операнда возвращают False, то возвращает False.
#Вывод:

В операторе AND оба операнда должны возвращать True, чтобы результат был True.
В операторе OR достаточно, чтобы хотя бы один операнд возвращал True, чтобы результат был True.

x = 5
y = 10
z = 15

# Проверяем, что все три числа больше 0
if x > 0 and y > 0 and z > 0:
    print("Все числа больше 0")

# Проверяем, что хотя бы одно число меньше 0
if x < 0 or y < 0 or z < 0:
    print("Хотя бы одно число меньше 0")



age = 25
income = 3000

# Проверяем, что возраст меньше 18 или доход меньше 2000
if age < 18 or income < 2000:
    print("Вы не подходите для этого предложения")
else:
    print("Вы можете принять участие")


В первом примере, с помощью оператора AND, мы проверяем, что все три числа x, y и z больше 0. Если это условие выполняется, то выводится сообщение "Все числа больше 0".

Во втором примере, с помощью оператора OR, мы проверяем, что либо возраст age меньше 18, либо доход income меньше 2000. Если хотя бы одно из условий выполняется, то выводится сообщение "Вы не подходите для этого предложения", иначе выводится "Вы можете принять участие".
